# N Largest CLI

An intuitive CLI to receive id, number pairs from a remote file and print the ids of the highest N numbers in the file.

## Usage

#### Main Command 
```
nlargest [OPTIONS] COMMAND [ARGS]
```

##### Options

* ```--help``` : display help information.
